2. 변수 선언

(변수 타입 종류)
- String a - 문자열 변수
- int - 정수형 변수
- double - 실수형 변수
- bool - 불리언 타입 변수
- num - 숫자형 변수
- dynamic - 모든 타입 변수 (수정 가능)
- final - 모든 타입 변수 (수정 불가능)
- const - 모든 타입 변수 (수정 불가능, 런타임 성능↑)
- var - 모든 타입 변수 (수정 불가능, 전역 변수)

(타입 변경)
- 변수 타입 변경 - a as "type" ; (type: num, dynamic...)
